# Dear Lola Kitchen & Bar

A premium, cinematic web experience for Dear Lola Kitchen & Bar. Built with React, Vite, Framer Motion, and Tailwind CSS.

## Features
- Cinematic Loading Screen with liquid expansion reveal.
- Magazine-style asymmetric layouts.
- Staggered menu with interactive hover states.
- Cinematic horizontal scroll gallery.
- AI-integrated chat assistant (Lola AI).
- Fully responsive design with premium micro-animations.

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS + Vanilla CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel (Standard SPA)

## Deployment Rules (Vercel)
- Framework Preset: Vite
- Output Directory: dist
- SPA Routing: Configured in `vercel.json`
- Build Verification: Produces `dist/index.html`
